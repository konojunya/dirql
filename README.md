# Description

dirql is a command line interface that searches by specifying a folder like SQL.

# Installation

install with npm or yarn

- npm

```
$ npm install -g dirql
```

- yarn

```
$ yarn add -g dirql
```

# Usage

```shell
$ dirql
$> select * from ./
```
